HTTP Method Path Description
GET / Displays the static home page.

Users

HTTP Methods Path Description
GET /users Retrieves all users from the database
GET /users/:id Retrieves a single user from the database
POST /users/register Adds a single user to the database
PATCH/PUT /users/update/:id Updates a single user
DELETE /users/delete/:id Deletes a single user from the database

Items

HTTP Method Path Description
GET /items Retrieves all items within the database
GET /items/:id Retrieves a single item from the database
POST /products/add Adds a single product to the database
PUT/PATCH /items/update/:id Updates a single product record
DELETE /items/delete/:id Deletes a single product from the database

Orders

HTTP Method Path Description
GET user/:userID/orders' Retrieves all orders
GET user/:userID/order/:orderID' Retrieves a single order from the database
POST /user/:userID/order To add an order
PUT/PATCH user/:userID/order/:orderID' Updates an order record
DELETE /user/:userID/order Deletes a single order from the database
DELETE user/:userID/order/:orderID Deletes all orders from the database
